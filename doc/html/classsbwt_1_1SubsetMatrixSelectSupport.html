Detailed Description

template<typename bitvector_t>
class sbwt::SubsetMatrixSelectSupport< bitvector_t >

A subset select support based on bitvector select support on indicator bitvectors for each character. This class does not own the bit vectors its pointing to. But it does own the select support data.
